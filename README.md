# zipcodeinfo
Node Module to provide the details for the Global Countries and their Zip Codes

## Usage

```bash
$ npm start US 38119
```
Sample Response
```json
{
    "post code": "38119",
    "country": "United States",
    "country abbreviation": "US",
    "places": [
        {
            "place name": "Memphis",
            "longitude": "-89.8501",
            "state": "Tennessee",
            "state abbreviation": "TN",
            "latitude": "35.0821"
        }
    ]
}
```
